2.O que é o serviço

O Jarvis Carrossel OS é um plugin para o Figma Desktop que monta carrosséis: aplica textos em lote, aprende a sua voz para reescrever roteiros, sincroniza perfil e cabeçalhos e gera imagens com IA. Algumas funções dependem de uma assinatura ativa; a geração de imagens consome créditos de IA, comprados à parte.
